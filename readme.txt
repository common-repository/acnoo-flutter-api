== Description ==

The plugin is used for config the Acnoo flutter mobile apps and support Rest API to connect to the app. The major function of this plugin is to communicate between the mobile app (android or iOS) and the WordPress website.

[youtube https://youtu.be/hTtrRJAEC20]

Acnoo focuses on building modern useful web templates as well as mobile application for solving modern problems. As a part of it, different mobile apps require functionality to communicate with websites. As a solution, this plugin Acnoo Flutter API is created. It is specially built for communication between the mobile application and the website using REST API.

## Our Great Features

We have created this one plugin to integrate with all our mobile applications. That is why this particular plugin is packed with features.
Here are some of the notable features of our plugin: 

- Custom 'Authentication' method, no third party plugin needed.
- Deliveryman API for deliveryman profile data, orders, order status updating.
- Deliveryman image API for getting and updating profile picture for deliveryman.
- WooCommerce order notes api, for customized message about the order.
- WooCommerce product list api, to get order id, date, title, price, product variations and what not.
- WooCommerce product category list api, to retrieve category id, title, slug etc.
- WooCommerce specific product data using product id.
- Getting WooCommerce product id, from product URL.
- Getting WooCommerce product category id from category URL.
- User management api, to get, create, update, delete user data.
- Sending push notifications using Firebase, using device token.
- Automatic push notification sending when order status is changed, e.g - processing to out for delivery.
- Dokan vendor registration from mobile app.
- Vendor and Customer social login using Google, Facebook account.
- Features selection, turning features on and off.
- Refresh Token API, better authentication for Vendor apps.

More of our features are coming soon. We are working actively on all our products, including flutter apps, WordPress plugins etc.

== Installation ==

= Manual Installation =

1. Upload the entire `/acnoo-flutter-api` directory to the `/wp-content/plugins/` directory.
1. Activate 'Acnoo Flutter API' through the 'Plugins' menu in WordPress.

= Better Installation =

1. Go to Plugins > Add New in your WordPress admin and search for 'Acnoo Flutter API'.
1. Click Install.
